# References

Kaggle notebook、discussion、外部記事などから得た知識を保存する場所です。

## 方針

- `knowledge/` は Git 管理する。Claude Code がいつでも読めるように、md で要約を残す。
- `raw/` は Git 管理しない。HTML、ipynb、スクリーンショット、取得ファイルなどの raw データをローカルに置く。
- raw を残す場合は `references/raw/` をローカルで作成する。
- 長い本文をそのまま貼らない。要点、使いどころ、リスク、実験候補に分けて要約する。
- 実験や提出に使った知識は、`SESSION_NOTES.md` や `submit/SUBMISSIONS.md` にも出典を書く。

## 保存先

- `knowledge/INDEX.md`: まず読む index。関連ファイルだけ開くための入口
- `knowledge/notebooks.md`: public notebook から得た知識
- `knowledge/discussions.md`: discussion から得た知識
- `knowledge/external_ideas.md`: 論文、記事、他コンペなどの外部知識
