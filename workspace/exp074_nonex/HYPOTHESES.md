# 仮説バックログ（2026-07-22 起点）

exp074 で「プールの 86.8% が単一パイロット」と分かったので、仮説を計測器・相手
モデル・ロジック・学習の4系統に整理し直した。締切は Simulation 8/16（残り約3.5週）。

採点欄: 影響=スコアに効く見込み / 費用=計算時間 / 状態。

H8 の中間結果が既に不穏です。`alakazam_dun` で
lucario_v2 では v030 が koff に +0.037 で勝っていたのに、
pub1034 では koff 0.645 / v030 0.595 と **−0.050 で符号が反転**しました。
これが全スロットで起きるなら、v030 の出荷判断そのものが計器の産物だったことになります。

C 系統に移ります。
