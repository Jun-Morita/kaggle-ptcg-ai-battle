# exp075 — デッキの見直し（silver 圏内が目標）

日付: 2026-07-22 開始
前提: exp074 で現行デッキ（Great Tusk/Crustle LO）の構造的限界が確定した。

## なぜデッキを替えるのか（証拠の要約）

1. **Alakazam（フーディン）系＝band重み41%に勝ち筋が無い**。攻撃は1試合1〜2回、
   ミル1.4〜2.4枚、サイド0枚。プールの0.847は相手の自滅産物（exp074 Stage 8）。
2. **H2/H1/H17 全棄却**＝方策でも小改造でも直らない。Superb Scissors は草エネ必須で
   永久使用不能、ミルは引き律速、エネ増量は band −0.0755。
3. 外部の独立報告とも一致: 「同じチェックポイントでもデッキでスコアが大きく変わる」
   （disc728071 で3人が独立に報告）。

## デッキに求める要件

- R1: Alakazam 系に勝ち筋がある（打てる・または妨害できる）
- R2: ミラーが弱すぎない（乗り換え先が人気ならミラーが最大マッチアップ。
      v025 の敗因 = Alakazam ミラー 0.32）
- R3: 手元に操縦できるパイロットがある（deck⊗pilot 結合、今年4回確認）
- R4: 壁（pure_wall 9.3%）と marnie(16%) に大負けしない

## Stage 1 スクリーン（n=100、粗ふるい）

各候補について:
1. **vs pub1034 stock**（=41%ブロブの代理。ミラー勝率、R2）
2. **vs v030 koff build**（現行に勝てるか）
3. **vs pure_wall**（AC crustle、R4）

注意: pub1034 は探索の決定化に Python random を使うため CRN 不成立（exp074）。
n=100 の SE≈0.05、n=200 でノイズ床±0.07 の実測。**大きな差だけ読む**。

## 発見（Stage 1 実行中に判明）

**E（exp054 で抽出済みの silver 帯実 Alakazam）≒ B（対我々 0-6 リスト）。**
差分は Dunsparce の版のみ: B=cardId 65(HP60) / E=cardId 305(HP70)。E が厳密上位互換。

つまり silver 帯で実際に勝っている Alakazam 型は「Enhanced Hammer x4 + Dunsparce x4 +
Enriching Energy + Battle Cage」変種で、pub1034 stock との差は
Xerosic x2 / Lillie's x1 / NZ x1 を抜いて上記を足した形。
→ B が S1 を通過したらビルドは E（HP70版）で行う。

   判定は S2 の band 加重に変更。事前登録の変更として明記）。
4. 全候補 S2 へ。A は計器の基準として続投。

## ミラー集中測定（n=400、パイロット固定）— デッキではミラーは直らない

exp077 で「Alakazam の穴はミラー1つだけ」と判明したので、そこだけ n=400 で測った。
パイロットを pub1034 に固定し、我々の60枚だけを変えて stock と対戦＝デッキ優位の分離。

```
stock (control)   0.495  (198-202)   ← 真値0.500、バイアス −0.005（較正良好）
E real-silver     0.490  (196-204)
C most-common     0.485  (194-206)
B hammer-hp60     0.492  (197-203)
```

**3つの実物リストすべてがコントロールと誤差範囲内（SE 0.025）。** Enhanced Hammer 仮説は棄却。

exp058 で方策側チューニングも NO-GO だったので、**ミラーの 0.333 はデッキでも方策でも
直せない** ＝ 実際の Alakazam 使いとの純粋な操縦技量差（exp074 のパイロット差 0.21-0.25 と同種）。

## 判断

- 乗り換えは **+0.030**（実ラダー実測ベース）。今日見つけた唯一の正のレバー。
- リストは **stock を使う**。E/C/B はミラーで差が無く、stock だけが実ラダーの
  実績（174試合 0.621）を持つ。測定済みの成果物から理由なく逸れない。
- eligible は **片枠のみ Alakazam**、もう片枠は koff を維持。
  理由: Alakazam シェアが増えるとミラーの穴が拡大するリスクのヘッジ、
  かつラダー上での A/B にもなる。
