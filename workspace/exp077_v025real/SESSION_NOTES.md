# exp077 — v025(Alakazam) と koff を実ラダーで直接比較

日付: 2026-07-22
動機: exp075 の乗り換え判断を、プール推定でなく**実戦データ**で行う。
乗り換え候補（pub1034 Alakazam）は v025 として 07-15 に一度提出済みで、
meta_watch スナップショットが2枚残っている。

## 提出IDの特定（推測でなく実測）

各リプレイ・ディレクトリから**我々自身の60枚**を抽出して照合:

```
0716_54728325  PUB1034-ALAKAZAM   (Telepath Psychic Energy x4, Dudunsparce x4, Abra x4)
0717_54728325  PUB1034-ALAKAZAM
0721_54797761  KOFF-LO            (Great Tusk x4, Dwebble x4, Crustle x4)
```

→ **sub 54728325 = v025 = pub1034 Alakazam** と確定。

two-proportion z = **+1.14**（n=174 vs 147）。有意水準には届かないが、
**実ラダー実測**である点でプール値（0.13楽観、86.8%単一パイロット）より遥かに信頼できる。

## 解釈

1. **Alakazam の穴はミラー1つだけ。** 他は全て koff 以上。特に koff の実戦最悪マッチ
   non_ex_attackers（0.412）を 0.588 で克服している。
2. **ミラーが最大スライス**（v025 の対戦の21%、koff では29%）。
   0.333 → 0.500 にできれば合計 ≈ 0.656。
3. **0.621 は「まだ収束していない」ことを示す。** 均衡に達した agent の勝率は 0.5 に寄る。
   174試合 0.621 を維持 ＝ 876.7 は到達点でなく通過点だった可能性が高い。
   koff の 0.558（147試合）は均衡に近い。

## exp075 への含意

乗り換えの是非は「ミラーを直せるか」に集約される。exp075 Stage 2 が測っている
silver帯実物リスト（Enhanced Hammer x4 + Dunsparce x4、= 対我々0-6の型）が
stock のミラーを改善するかが判断材料。

既知の負の結果: exp058 で pub1034 のミラー重みチューニングは NO-GO
（n=300: 0.487/0.513/0.483）。**方策側では直らない**ので、デッキ側で直す必要がある。
これは v012（デッキ比率修正が成功、方策修正は失敗続き）と同じ構図。

## 注意点

- 期間が違う（v025: 07-16/17、koff: 07-21）。メタは動く。
- レート帯が近い（v025 ~876-911、koff ~886）ので圧縮バイアスは概ね共通。

  リロール1サイクル分であり、24日残っているので許容範囲。
