# exp079 — Alakazam ミラーに「見落とし機構」はあるか（無かった）

日付: 2026-07-23

## 動機

exp077 で、Alakazam への乗り換えは **ミラー以外で +0.091 / ミラーだけで −0.061** と判明。
ミラーを 0.333 → 0.500 にできれば +0.061 で、乗り換え自体の +0.030 の2倍以上。

修正手段のうち2つは既に閉じている:
- 方策の重みチューニング: exp058 で NO-GO（n=300）
- デッキ変更: exp075 ミラー掃引で NO-GO（n=400、実物3リストがコントロール0.495と誤差内）

しかし今年成功した修正はどれも第3の型＝**方策が実在機構を見ていない**（v010 gust /
v011 revenge / v023 KO_OFF の機能削除 / v030 述語の誤答）。ミラーでこの型は未探索だった。

## 方法

v025 がミラーで戦った36試合（12W-24L）を使用。リプレイには**相手側の観測と行動が両方入っている**
ので、相手の局面を pub1034 の方策に食わせ、同一局面・同一合法手での選択を突き合わせた。
判定は exp022 の take-when-legal:
- exposure差が大きく take率が同じ → throughput/引きの差、修正不能
- exposure正常で take率だけ低い → ゲートされた意思決定の漏れ（gust型）、修正価値あり

## 結果: 漏れは無かった

決定 2232件（36試合）:

```
option      legal  THEM%   US%     gap
END          1247   5.1%   7.5%   −2.4%
PLAY         1103  59.7%  65.8%   −6.1%
ATTACK        716   9.2%  14.8%   −5.6%
RETREAT       693   4.2%   0.6%   +3.6%
ATTACH        528  30.5%  25.8%   +4.7%
EVOLVE        453  15.9%  27.2%  −11.3%
ABILITY       297  21.2%  19.5%   +1.7%
```

## 途中で犯した誤りと訂正（2件）

**誤り1: n=6 の予備結果で ABILITY +22.6% を「大きな信号」と報告した。**
36試合では **+1.7%** に収束＝ノイズだった。小標本の差を確認前に有望と述べたのは不適切。

**誤り2: ctx4/5/8 の「一致 0.000」を系統差の候補として扱った。**
実例を見ると相手の記録行動が `[]`（空）で `minCount=1`＝**記録の欠落**。行動差ではない。

誤り2は MAIN の数字も汚染していた:
```
THEM take合計 1114 / legal 1247 → 133件(10.7%)が空記録
US   take合計 1247 / legal 1247 → 空なし
```
相手の全take率が約11%過小評価。補正後:

| option | THEM(補正) | US | gap |
|---|---|---|---|
| PLAY | 66.9% | 65.8% | +1.1% |
| EVOLVE | 17.8% | 27.2% | −9.4% |
| ATTACK | 10.3% | 14.8% | −4.5% |
| ABILITY | 23.7% | 19.5% | +4.2% |
| RETREAT | 4.7% | 0.6% | +4.1% |

補正後も **gust型の漏れは無い**。最大でも ABILITY/RETREAT の +4% 程度で、
v010 の「Boss 0回/試合 → 1回/試合」のような構造的欠落とは桁が違う。

## 結論

**Alakazam ミラーは、我々が持つ3つの手段（重み調律・デッキ変更・機構発見）すべてで直せない。**
残る傾向は「我々は進化と攻撃を急ぎ、相手は待って引く」というスタイル差だが、
exp013 で「上位の決定を模倣しても勝率は上がらない」と既に確認済み（v009 が 0.500 で終わった）。
これを追う根拠はない。

ミラーの 0.333 は、実際の Alakazam 使いとの**純粋な操縦技量差**として受け入れる。
exp074 が測った 0.21〜0.25 のパイロット差と同じもの。

複数人が独立に報告）。我々の RL ネガティブ16件との違いは**教師が上位プレイヤーか否か**。
